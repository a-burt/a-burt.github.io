21 Century Learning Rubric Tool

Use this tool to determine how well your class activity teaches 21st Century skills.

Welcome

This tool is designed to provide feedback about the use of 21st Century skills in a single class activity. It will ask you questions about the skills of collaboration, knowledge construction, real-world problem-solving and innovation, use of ICT for learning, self-regulation and skilled communication. At the end of the questions it will show you your ranking for each skill.
